# Monte Hall problem

Simulates the Monte Hall problem using Clojure.

Inspired by a stranger who also wrote a program to understand this.

## Installation

1. Clone or download this repository.
2. Install Leiningen (if you haven't already)

## Usage

Run using Leiningen

    lein run
